# Giriraj Portfolio (https://giriraj-portfolio-7vgt.vercel.app) 

A minimal, responsive developer portfolio built with Next.js and Tailwind CSS, inspired by Brittany Chiang's design.

## Features

- Fixed sidebar navigation on desktop
- Smooth scrolling between sections
- Dark theme with minimal typography
- Fully responsive design
- Active section highlighting
- Hover effects and transitions

## Sections

- About
- Work/Experience
- Projects
- Services (Freelance)
- Skills
- Contact

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

Edit the content in `app/page.tsx`:

- Update your name and title in the header section
- Modify the about section with your bio
- Add your work experience
- Showcase your projects
- List your services
- Update your skills
- Change contact information
- Update social media links (GitHub, LinkedIn, Twitter)

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- React Compiler

## Deployment

Deploy easily with Vercel:

```bash
npm run build
```

Or deploy to Vercel with one click from your repository.
